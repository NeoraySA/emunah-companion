# Scripts

Utility scripts for development, deployment, and maintenance.

## Planned Scripts

- `setup-db.sh` – Create database and run initial migrations
- `generate-encryption-key.ts` – Generate a secure AES-256 key for journal encryption
- `cleanup-gcs.ts` – Remove orphaned GCS objects (soft-deleted >30 days)
- `export-content.ts` – Export all content/translations to JSON
